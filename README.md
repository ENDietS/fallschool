# ENDietS Gather to Nourish Fall School Website

This folder contains the multi-page website for the ENDietS Gather to Nourish Fall School 2026.

Open `index.html` in Chrome to preview locally.

Main pages:
- `index.html` — Home
- `welcome.html` — ENDietS Presidents' Welcome Message
- `about.html` — Fall School 2026
- `team.html` — ENDietS Team
- `programme.html` — Programme
- `registration.html` — Registration Info
- `venue.html` — Venue
- `faq.html` — FAQ

To publish, upload the full folder or ZIP to a static hosting service such as Netlify.
